  ```bash
  # SSH key is in ORACLE_SSH_KEY env var with spaces instead of newlines — fix with Node.js:
  node -e "
  const raw = process.env.ORACLE_SSH_KEY || '';
  const b64 = raw.replace('-----BEGIN OPENSSH PRIVATE KEY-----','').replace('-----END OPENSSH PRIVATE KEY-----','').replace(/\s+/g,'');
  const wrapped = b64.match(/.{1,70}/g).join('\n');
  require('fs').writeFileSync('/tmp/oracle_key', '-----BEGIN OPENSSH PRIVATE KEY-----\n' + wrapped + '\n-----END OPENSSH PRIVATE KEY-----\n');
  "
  chmod 600 /tmp/oracle_key
  ```

  ## 4. How to Push Code from Replit (IMPORTANT)

  **Bash `curl` to GitHub API with auth often times out** from Replit containers. Use the `code_execution` sandbox instead — it reads the token directly from Replit's secrets store:

  ```javascript
  const vars = await viewEnvVars();
  const token = vars.envVars.shared['GITHUB_TOKEN'];
  // token is now valid — use in fetch() calls to api.github.com
  ```

  **6-step tree commit pattern** (SKILLS.md Rule 3):
  1. `GET /repos/{repo}/git/ref/heads/main` → headSha
  2. `GET /repos/{repo}/git/commits/{headSha}` → treeSha
  3. `POST /repos/{repo}/git/blobs` for each file → blobSha(s)
  4. `POST /repos/{repo}/git/trees` with base_tree + new blobs → newTreeSha
  5. `POST /repos/{repo}/git/commits` with tree + parent → newCommitSha
  6. `PATCH /repos/{repo}/git/refs/heads/main` with newCommitSha
