## 9. JazzDrive Rules (NEVER Break These)

```
✅ JazzDrive = VIDEO streaming and downloads ONLY
❌ NEVER use JazzDrive for images (backdrops, thumbnails)
✅ All images = TMDB API > IMDB > OMDB API > Google > Youtube  (free, fast, has everything)
✅ JazzDrive fallback for posters = ONLY if user do not have internet package then use as fallback if user have net then do not use (poster comes free)

Link generation critical notes:
- Extract shareKey from URL with regex: /\/(?:share-landing\/f|share\/f|f)\/([^\/?#]+)/
- Step 1: POST /sapi/link/login?action=login → get validationkey + JSESSIONID cookie
- Step 2: GET /sapi/media/video?action=get&shared=true&key=SK&validationkey=VK + Cookie
- Step 3: Build URL = record.url + "?filename=" + encodeURIComponent(record.name)
- NEVER append validationkey= to final URL — k= token is self-signing, adding vk breaks it
- Cache TTL: 6 hours (same link reused for watch + download)
```
