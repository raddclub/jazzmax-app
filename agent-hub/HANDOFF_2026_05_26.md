# Step 1: Decode SSH key (env var has embedded spaces — strip them first)

import os, base64
raw = os.environ['ORACLE_SSH_KEY']
# Key may have spaces instead of newlines — reconstruct properly
raw = raw.strip()
if ' ' in raw:
    header = '-----BEGIN OPENSSH PRIVATE KEY-----'
    footer = '-----END OPENSSH PRIVATE KEY-----'
    body = raw.replace(header, '').replace(footer, '').strip()
    body_clean = '\n'.join([body[i:i+64] for i in range(0, len(body.replace(' ','')), 64)])
    key = header + '\n' + body_clean + '\n' + footer + '\n'
else:
    key = base64.b64decode(raw).decode()
with open('/tmp/oracle_key','w') as f: f.write(key)
import subprocess; subprocess.run(['chmod','600','/tmp/oracle_key'])
print('Key written to /tmp/oracle_key')
"

## 2. GitHub API Rules (CRITICAL)

**The repo is public.** Use raw GitHub for reading, API for writing.

```python
import urllib.request, base64, json, os
TOKEN = os.environ['GITHUB_TOKEN']

# READ a file
def gh_read(path):
    url = f"https://api.github.com/repos/raddclub/raddflix-app/contents/{path}"
    req = urllib.request.Request(url, headers={"Authorization": f"token {TOKEN}"})
    j = json.loads(urllib.request.urlopen(req).read())
    return base64.b64decode(j["content"]).decode("utf-8"), j["sha"]

# WRITE a file  
def gh_write(path, content, sha, message):
    url = f"https://api.github.com/repos/raddclub/raddflix-app/contents/{path}"
    payload = json.dumps({
        "message": message,
        "content": base64.b64encode(content.encode()).decode("ascii"),
        "sha": sha
    }).encode()
    req = urllib.request.Request(url, data=payload, method="PUT",
        headers={"Authorization": f"token {TOKEN}", "Content-Type": "application/json"})
    r = json.loads(urllib.request.urlopen(req).read())
    return r["commit"]["sha"]
```

- **Always fetch current SHA** before writing (GitHub rejects writes with wrong SHA)  
- **Never use `git` shell commands** — main agent cannot push directly  
- **Never use `raw.githubusercontent.com`** for private paths (repo was private until Session 5)

## 6. JazzDrive Rules (NEVER break)

```
✅ JazzDrive = VIDEO links ONLY
❌ NEVER use JazzDrive for images (posters, thumbnails, banners)
✅ Images: TMDB API → IMDB → OMDB → Google → YouTube

Link generation (jazzdrive_service.dart):
  Step 1: POST https://cloud.jazzdrive.com.pk/sapi/link/login?action=login
          Body: { "shared": true, "key": shareKey }
          → returns { validationkey, Set-Cookie: JSESSIONID }
  Step 2: GET  https://cloud.jazzdrive.com.pk/sapi/media/video?action=get
               &shared=true&key=shareKey&validationkey=VK
          Header: Cookie: JSESSIONID=...
          → returns record with { url, name }
  Step 3: Final URL = record.url + "?filename=" + encodeURIComponent(record.name)
  
  ⚠️  NEVER append validationkey= to final URL — k= token is self-signing
  shareKey regex: /\/(?:share-landing\/f|share\/f|f)\/([^\/?#]+)/
  Cache: 6 hours in memory + stream_cache SQLite table
```
