📖 Help & Guide

🤖 AI-Powered Site Routing

The AI classifies each movie and picks the best site(s) to search:

CategorySites Used (in order)
English / HollywoodVegaMovies → KatMovieHD → WatchMoviesPK → SSRMovies
Indian / Bollywood / South / UrduRogMovies → WatchMoviesPK → SSRMovies → VegaMovies
AnimeGokuHD → PikaHD → VegaMovies → KatMovieHD
Animated (Disney/Pixar)VegaMovies → KatMovieHD → WatchMoviesPK → GokuHD → PikaHD
Mixed / UnknownAll sites in default order

Order: GROQ → Gemini → Keyword fallback (always works).

v4.0 fast path: if you tag a language (e.g. Indian, English, Anime) or your TMDB key resolves the original language, the AI call is skipped and routing happens instantly.

🏷 Search Tags & Punctuation NEW v4.0

Tags are optional but help the matcher reject wrong results (e.g. searching Agent 2023 and getting Agent Elvis).

Format: Title (YEAR) Language

Agent (2023) Indian Spider-Man (2002) English Sherlock Holmes [2009]; English KGF 2 Telugu Naruto Anime Premalu (2024) ← year only, no lang The Dark Knight ← untagged still works

Recognised languages: indian, hindi, telugu, tamil, malayalam, kannada, punjabi, english, anime, animated, korean, japanese.

Punctuation tolerated: ( ) [ ] { } ; , : . & ", smart quotes, and en/em-dashes are auto-cleaned before searching.

🛡 Wrong-Match Protection NEW v4.0

Every result URL is checked by a 3-signal relevance gate before downloading:

  • Coverage — at least 70 % of your title words must be in the result URL.
  • Pollution — extra title words like Elvis, No Way Home, etc. are rejected. One-word queries get zero tolerance.
  • Year — if you (or TMDB) supply a year, the URL must contain it ±1.

If a site returns the wrong title, the race quietly falls through to the next site instead of downloading garbage.

🔑 Setting Up AI Keys

AI keys are optional but recommended. Without them, a keyword-only fallback is used.

  • Open Settings → AI API Keys
  • Paste your GROQ key (free at console.groq.com/keys) and/or Gemini key (free at aistudio.google.com/apikey)
  • Click Save, then Verify to confirm the key works

🎬 Setting Up Metadata Keys (TMDB / OMDB) NEW v4.0

Both keys are optional — the app works without them. Adding them improves match accuracy by looking up the canonical title, year, and original language for every search.

  • TMDB (primary) — free at themoviedb.org/settings/api
  • OMDB / IMDB (fallback) — free at omdbapi.com/apikey.aspx
  • Open Settings → 🎬 Metadata API Keys, paste, Save, Verify.

Looked-up data is cached on disk for 30 days, so the same query won't hit the API twice.

➕ Adding Movies

  • Type or paste movie names — one per line or comma-separated
  • Add 1 or 100s at once — they all go into the queue
  • Pick AI (Auto) to let the AI choose the best site
  • Use the new tag format (above) for stricter matching when needed

⚡ Queue Controls

  • ⏸ Pause: Suspend an active job
  • ▶ Resume: Continue a paused job
  • ✕ Cancel: Stop the job immediately
  • ↺ Retry: Re-queue a failed/cancelled job
  • 🗑 Remove: Delete the job from the list
